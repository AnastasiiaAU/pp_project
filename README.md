## Исследование потребителей аптечной сети 36.6
Дата реализации: май 2024

В исследовании принимали участие студенты ВШЭ. В рамках работы были реализованы:
- Подготовка и валидация данных;
- Корреляционная матрица;
- Регрессионные модели по выделенным гипотезам исследования;
- Разработаны рекомендации по улучшению клиентского сервиса.
